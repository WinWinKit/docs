WinWinKit is a platform designed to provide an easy way to set up and launch referral programs in mobile apps.
With WinWinKit, you can create referral programs of varying complexity, manage rewards with just a few clicks, and launch experiments (soon).

## Quickstart

The first step to start using WinWinKit is to create an account, configure a referral program and begin integration via SDK or REST API.
