<Warning>
  Integration with [RevenueCat](https://www.revenuecat.com) is required to use
  this reward type. <br />
  Log in to your project in [WinWinKit](https://app.winwinkit.com), click Settings
  -> Integrations -> RevenueCat -> Connect and follow the instructions to enable
  it.
</Warning>

## Examples

- **Feature Access**: Grant access to a feature as a reward for engaging with referrals.
- **Premium Access**: Grant access to premium features for a limited time.

Learn more about [RevenueCat Entitlements](https://www.revenuecat.com/docs/getting-started/entitlements).
